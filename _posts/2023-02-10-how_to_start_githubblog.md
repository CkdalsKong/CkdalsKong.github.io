## 1. 원하는 테마 찾기

- 찾은 테마를 fork 하여 repositoryname을 내 깃허브 아이디.github.io로 바꾸기
- _config.yml 파일에서 url과 title 변경하기

## 2. 포스팅 시작하기

- Add file -> Create new file
- _posts/ 를 작성해 posts 파일 생성하고, md 파일을 넣는다.

## 3. 개발환경 설치

- github에서 repository를 github desktop에 clone하기
