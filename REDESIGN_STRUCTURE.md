## 색상 & 디자인 토큰

```
배경 계층:
  L0 (최하): #0d0f14  ← 전체 페이지 배경
  L1 (카드): #13161e  ← 카드, 섹션 배경
  L2 (호버): #1a1f2e  ← 카드 hover 상태
  Border:    #1e2433  ← 구분선

텍스트:
  Primary:   #e2e8f0  ← 본문
  Secondary: #94a3b8  ← 보조
  Muted:     #64748b  ← 날짜, 메타

포인트 색상:
  Indigo:    #6366f1  ← 주 강조 (버튼, 링크, dot)
  Sky:       #38bdf8  ← 보조 강조 (태그, 배지)
  Gold:      #f59e0b  ← Top venue 강조 (ICML 배지)
  Green:     #10b981  ← 진행 중 항목

폰트 크기 스케일:
  xs:   0.75rem  (11px) ← 배지, 메타
  sm:   0.875rem (14px) ← 보조 텍스트
  base: 1rem     (16px) ← 본문
  lg:   1.125rem (18px) ← 섹션 텍스트
  xl:   1.25rem  (20px) ← 소제목
  2xl:  1.5rem   (24px) ← 섹션 제목
  3xl:  2rem     (32px) ← 이름
  4xl:  2.5rem   (40px) ← Hero 큰 텍스트
```
