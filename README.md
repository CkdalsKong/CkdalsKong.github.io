# changmin-lee.com

Personal academic homepage of Changmin Lee, built with [Academic Pages](https://github.com/academicpages/academicpages.github.io) and hosted on GitHub Pages.

## Local development

```bash
bundle install
bundle exec jekyll serve -l -H localhost
```

Then open http://localhost:4000.

## Deployment

Push to the `main` branch of `<your-github-id>.github.io` repository. GitHub Pages will build and deploy automatically.

## Custom domain

DNS records for `changmin-lee.com` point to GitHub Pages IPs. See the `CNAME` file in the repo root.
